# Product Terms

## Loop Engineering

Loop engineering is the practice of turning product or engineering intent into
observable agent loops. A loop routes the work into the right project context,
runs staged agent phases, preserves evidence, feeds failures back into the next
attempt, and ends as `done` or `blocked`.

Use this term when describing the product category, public positioning, or
operator-facing workflow.

## Workflow Loop

A workflow loop is one concrete run through the configured phases, usually
brainstorm, plan, implement, test, review, and report. Retryable failures feed
back into implementation; unclear or unsafe work pauses for a human operator.

## Operator Checkpoint

An operator checkpoint is a human review point where devos.ing exposes logs,
run state, PR context, blockers, and verification evidence before work continues
or closes.
