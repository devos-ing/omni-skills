# Company Analysis

Install:

```bash
npx omniskill@latest install company-analysis
```

Invoke `$company-analysis` with a company or ticker, horizon, research question, and as-of expectation. The workflow returns business-model, competitive-position, filing, management-claim, event, and limitation evidence for a parent coordinator or standalone user. It uses host-provided browsing and public sources; it requires no API key and does not issue personalized trade advice.
