# Financial Analysis

Install:

```bash
npx omniskill@latest install financial-analysis
```

Invoke `$financial-analysis` with a company or ticker, horizon, and available filings. The workflow returns revenue, margins, cash flow, balance-sheet quality, accounting signals, trend consistency, and limitations for a parent coordinator or standalone user. It uses host-provided browsing and public sources; it requires no API key and does not issue personalized trade advice.
