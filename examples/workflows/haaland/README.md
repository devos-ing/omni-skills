# Haaland GetSuperpower

Use this workflow when an agent should make a playful Haaland/JTS meme: a compact
football-finisher joke, social caption, or parody post concept with a clear
punchline.

The included meme logo is an original parody asset at
`skills/haaland/assets/haaland-meme-logo.svg`. It is not an official player,
club, league, or sponsor mark.

Install it from the repo root:

```bash
bun run dev -- install examples/workflows/haaland
```

Validate it while authoring:

```bash
bun run dev -- validate examples/workflows/haaland
```

Refresh the checked skill fingerprints after editing the local meme skill:

```bash
bun run dev -- lock examples/workflows/haaland
```
