# Macro Analysis

Install:

```bash
npx omniskill@latest install macro-analysis
```

Invoke `$macro-analysis` with a geography, asset scope, horizon, and as-of expectation. The workflow returns growth, inflation, policy, liquidity, event-calendar, scenario-implication, and limitation evidence for a parent coordinator or standalone user. It uses host-provided browsing and public sources; it requires no API key and does not issue personalized trade advice.
