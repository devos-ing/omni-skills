# Market Structure

Install:

```bash
npx omniskill@latest install market-structure
```

Invoke `$market-structure` with a market or index, horizon, and available breadth and volatility evidence. The workflow returns breadth, volatility, concentration, positioning-proxy, technical-confirmation, and limitation evidence for a parent coordinator or standalone user. It uses host-provided browsing and public sources; it requires no API key and does not issue personalized trade advice.
