# Rates Analysis

Install:

```bash
npx omniskill@latest install rates-analysis
```

Invoke `$rates-analysis` with a geography, horizon, and available curve and credit evidence. The workflow returns yield-curve, real- and nominal-rate, credit, transmission, scenario-implication, and limitation evidence for a parent coordinator or standalone user. It uses host-provided browsing and public sources; it requires no API key and does not issue personalized trade advice.
