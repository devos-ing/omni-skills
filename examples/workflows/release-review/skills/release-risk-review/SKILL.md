# Release Risk Review

Use this skill when a workflow step needs a release-focused risk check before
implementation begins.

Review the request for:

- rollout scope and sequencing
- rollback path and owner confirmation
- migration, data, or configuration risk
- user-visible blast radius
- evidence needed before and after release

Ask for amendments when a release change lacks rollback steps, observable
success criteria, or a clear owner decision gate.
