# Risk Analysis

Install:

```bash
npx omniskill@latest install risk-analysis
```

Invoke `$risk-analysis` with a synthesized draft, source list, and decision horizon. The workflow returns contradictions, missing evidence, failure modes, triggers, invalidation, and a verification verdict for a parent coordinator or standalone user. It uses host-provided browsing and public sources; it requires no API key and does not issue personalized trade advice.
