# Valuation Analysis

Install:

```bash
npx omniskill@latest install valuation-analysis
```

Invoke `$valuation-analysis` with a company or ticker, horizon, and available price and financial evidence. The workflow returns explicit base, bull, and bear assumptions, sensitivity, expectation risk, catalysts, and limitations for a parent coordinator or standalone user. It uses host-provided browsing and public sources; it requires no API key and does not issue personalized trade advice.
